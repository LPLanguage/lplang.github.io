Download LPL

Welcome to the LPL download page. Here, you can find the latest versions of the LPL language module for your projects.

Installation Instructions

1. Download the LPL Module

Choose the version of the LPL module that suits your needs:

  • Download LPL 1.0 Module
  • Download LPL 1.1 Module
  • Download LPL 2.0 Module
  • Download LPL 3.0 Module(demo)

2. Include the Module in Your HTML(js)

Once you've downloaded the module, include it in your HTML file by adding the following script tag to the head section of your HTML:

<script src="path/to/selected-lpl-module.js"></script>

3. Start Writing LPL Code(js)

You can now start writing LPL code within your HTML by using the <lpl> tags. For LPL 1.0, you can use the var code = ``; variable located in the LPL 1.0 module.js file.

Example

<html>
    <head>
        <script src="path/to/selected-lpl-module.js"></script>
    </head>
    <body>
        <h1>My LPL Page</h1>
        <lpl>
            print("Hello, LPL!");
        </lpl>
    </body>
</html>
