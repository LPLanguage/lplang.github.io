Getting Started

If you are new to LPL, here's how you can get started:

  1. Download the LPL module for your preferred version.
  2. Include the module in your project by adding the script tag:
    3. <script src="LPL-Module.js"></script>
  3. Begin writing code in the <lpl> tag to harness the power of LPL.
